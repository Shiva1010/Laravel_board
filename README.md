# Download
* `git clone https://github.com/Yu-hanCheng/Laravel_board.git`
# Settings
* login to mysql and CREATE database
* Edit .env.example to .env, and set DB connection
# Steps
1. `composer install` 
2. `php artisan migrate`
3. `php artisan key:generate`
4. `php artisan serve`
